# Day 4: Decision Tree Classifier — Iris Dataset

## Overview
Implementation of Decision Tree classification on the Iris dataset
with comprehensive model evaluation using confusion matrix, 
precision, recall, and F1 score.

## Key Learnings
- Decision Tree algorithm with Gini/Entropy splitting
- Overfitting control via max_depth and min_samples_split
- Confusion Matrix interpretation
- Precision vs Recall trade-off

## Results
| Metric    | Score  |
|-----------|--------|
| Accuracy  | 96.67% |
| Precision | 0.97   |
| Recall    | 0.97   |
| F1 Score  | 0.97   |

## Tech Stack
Python 3.10 | scikit-learn | matplotlib | numpy

## How to Run
```bash
pip install scikit-learn matplotlib numpy
python day4_decision_tree.py
```
